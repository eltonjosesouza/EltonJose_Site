# /social-01-setup

1. **Identify Context**: Determine the active blog post directory.
2. **Execute Script**: Run the helper script to generate the JSON file.

   ```bash
   python .agent/scripts/social/01_create_sidecar.py .
   ```

   _(Ensure you run this terminal command in the directory found in step 1, or pass the absolute path as an argument)_

3. **Open File**: Open the newly created `social.json` file for the user to edit.
